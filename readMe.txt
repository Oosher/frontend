## Introduction
The Project Store Website is a comprehensive e-commerce platform designed to showcase and sell your project-related products. It offers a wide range of features to enhance the user experience, including user accounts, an admin panel, intuitive navigation, and more.

## Features
- **User Dashboard:** Manage user settings, view order history, and access the favorites and cart features.
- **Admin Panel:** Create, edit, and delete products, manage orders, and oversee the store's operation.
- **Navigation Bar:** Automatically displays six categories and provides access to the full category list in the left sidebar. Also includes user options, cart, and favorites.
- **Product Pages:** Allow users to view detailed product information, add items to their favorites, and add items to the cart.
- **Search Functionality:** Enables users to search for specific products or categories and reset searches.
- **Log In and Registration:** Secure user authentication system.
- **Favorites:** Users can mark products as favorites for quick access.
- **About Us:** Provides information about the project and the team.



Usage
    User Dashboard
        Log in or register to access your user dashboard.
        Change user settings.
        View and manage your orders.
        Add and remove items from your cart.
        Add items to your favorites.
    Admin Panel
        If you have admin privileges, access the admin panel.
        Create, edit, and delete products.
        Manage orders and oversee store operations.
    Navigation Bar
        Navigate the website easily using the top navigation bar.
        Access six automatic categories and the full category list in the left sidebar.
        Click on your avatar to open the user options menu.
        Monitor your cart and liked items.
    Product Pages
        Click on a product to view detailed information.
        Add items to your favorites or cart.
    Search Functionality
        Search for specific products or categories using the search bar.
        Reset your search by clicking the "x" button.
    About Us
        Learn more about our project and the team by clicking the "About Us" button in the footer.
